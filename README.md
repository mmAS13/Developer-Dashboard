# Developer Dashboard

A modern, responsive, and interactive personal dashboard for developers. It combines real-time data, productivity tools, and API-driven content into a sleek interface.

# Features

Clock & Date – Real-time digital clock with dynamic date display.
Weather Widget – Fetches and displays local weather using a public API.
Todo List – Add, edit, and remove tasks; persists data in local storage.
Quote of the Day – Random inspirational or programming quotes via API.
GitHub Profile Info – Displays user profile details (repos, followers, stats) dynamically using the GitHub API.

 # Technologies & Skills

HTML, CSS, Vanilla JavaScript
API Integration – GitHub, Weather, and Quotes APIs
Responsive Layout Design – Works across all screen sizes
Widget System – Modular and extendable dashboard components
Local Storage Management – Persist user preferences and tasks

# Next-Level Features (Optional Enhancements)

Dynamic Light/Dark Mode toggle
Drag-and-drop widget arrangement
Smooth animations for interactive elements
Customizable dashboard layout saved in local storage




# How It Works

Clock & Date: Uses JavaScript Date object for live updates.

Weather Widget: Fetches weather data using the OpenWeatherMap API
Todo List: CRUD operations are handled in JavaScript with local storage persistence.
Quote of the Day: Fetches random quotes via Quotes REST API
GitHub Info: Fetches user repositories, followers, and other stats from the GitHub API.
